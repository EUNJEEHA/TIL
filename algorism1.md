# 알고리즘 특강

## 1. 자료구조와 알고리즘 소개

> 자료구조 == 요리 재료
>
> 알고리즘 == 요리 방법

### 1-1. 자료구조의 종류

* 단순 자료 구조 -> 파이썬
  * 정수
  * 실수
  * 문자
  * 문자열
* 선형 자료 구조 -> __암기!__
  * 리스트
  * 스텍
  * 큐
* 비선형 자료 구조 -> __암기!__
  * 트리
  * 그래프
* 파일 자료 구조 -> 파이썬
  * 순차 파일
  * 색인 파일
  * 직접 파일
